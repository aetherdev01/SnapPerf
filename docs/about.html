Apa itu SnapPerf?

SnapPerf adalah modul Magisk yang dirancang khusus untuk perangkat Android bertenaga Snapdragon.

Berbeda dari tweak umum, SnapPerf menargetkan arsitektur unik Qualcomm untuk memberikan peningkatan yang nyata.

Persyaratan

  • Perangkat Android yang di-root (Magisk v24+ atau KernelSU)
  • SoC Qualcomm Snapdragon
  • Android 10 atau lebih baru direkomendasikan
  • Minimal 3 GB RAM

Cara Install

  1. Unduh file .zip dari halaman rilis.
  2. Buka Magisk Manager atau KernelSU Manager.
  3. Ketuk "Modul" lalu "Instal dari penyimpanan".
  4. Pilih zip yang diunduh dan reboot.
