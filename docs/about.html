What is SnapPerf?

SnapPerf is a Magisk module specifically engineered for Snapdragon-powered Android devices. It applies a curated set of kernel and system-level tweaks to improve overall performance, reduce latency, extend battery life, and enhance display smoothness.

Unlike one-size-fits-all tweaks, SnapPerf targets Qualcomm’s unique architecture including Kryo CPU clusters, Adreno GPU, and the Hexagon DSP to deliver meaningful, measurable improvements.

Requirements

  • Rooted Android device (Magisk v24+ or KernelSU)
  • Qualcomm Snapdragon SoC
  • Android 10 or later recommended
  • At least 3GB RAM

How to Install

  1. Download the .zip from the releases page.
  2. Open Magisk Manager or KernelSU Manager.
  3. Tap “Modules” → “Install from storage”.
  4. Select the downloaded zip and reboot.
